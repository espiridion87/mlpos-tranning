# MLOps Learning Program (12-Week Syllabus)

A structured learning path for DevOps engineers transitioning to MLOps roles. This program combines ML fundamentals with hands-on MLOps practice for deploying, monitoring, and maintaining ML systems in production.


---

## Week 1 – ML Basics I: Introduction to Models

* **Learning Focus**: What is a model, loss functions, gradient descent, hyperparameters
* **Estimated Time**: 115 min
* **Google ML Crash Course**:
  * [Linear Regression](https://developers.google.com/machine-learning/crash-course/linear-regression) (80 min)
  * [Logistic Regression](https://developers.google.com/machine-learning/crash-course/logistic-regression) (35 min)

## Program Summary

### Learning Outcomes

Upon completion, participants will be able to:

* Understand ML model fundamentals and evaluation metrics
* Design and implement data pipelines for ML
* Deploy ML models using containerization and modern serving patterns
* Implement CI/CD pipelines specifically designed for ML workflows
* Monitor models in production and detect drift
* Implement model governance and responsible AI practices
* Scale ML infrastructure efficiently

### Prerequisites

* Strong DevOps background (CI/CD, Docker, Kubernetes)
* Basic Python programming
* Familiarity with cloud platforms (GCP, AWS, or Azure)
